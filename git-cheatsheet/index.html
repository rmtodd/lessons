Git Cheatsheet

The commands you actually use, day to day.

Setup
git initInit a new repo in current dir
git clone <url>Clone a remote repo
git config --global user.nameSet your name globally
git config --global user.emailSet your email globally
Everyday workflow
git statusShow changed files
git add <file>Stage a file
git add .Stage all changes
git commit -m "msg"Commit staged changes
git pushPush commits to remote
git pullFetch + merge from remote
Branches
git branchList local branches
git branch <name>Create a branch
git switch <name>Switch to a branch
git switch -c <name>Create + switch in one step
git merge <branch>Merge branch into current
git branch -d <name>Delete a branch
Inspecting history
git log --onelineCompact commit history
git diffUnstaged changes
git diff --stagedStaged changes
git show <hash>Show a specific commit
git blame <file>Who changed what line
Undoing things
git restore <file>Discard unstaged changes
git restore --staged <file>Unstage a file
git revert <hash>New commit that undoes one
git reset --soft HEAD~1Undo last commit, keep changes
git stashShelve uncommitted changes
git stash popRestore stashed changes
Remote repos
git remote -vList remotes
git remote add origin <url>Add a remote
git fetchFetch without merging
git push -u origin <branch>Push + track remote branch
git push --force-with-leaseSafe force push
